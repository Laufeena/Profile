What is HTML?

HTML is the basic computing language that allows code to appear as readable text on a website to an online audience.

What is CSS?

CSS is what makes webpages look more like a webpage than a plain text file in a notepad document.

So what do we need to know about HTML?

Well firstly lets discuss HTML structures.

We have 3 main HTML sections:

Sections HTML tags Descriptions
head element <head>
</head>		 This is the top section of the HTML layout, this is where you put links to use libraries, css stylesheets and absolute links.
body element <body>
</body>		 This is the main area for the HTML structure. This is where all the main data that's displayed on the website is put.
footer element <footer>
</footer>		 This is where you would usually find all the website contact details for a company. You may also find policy links and site map information here.
